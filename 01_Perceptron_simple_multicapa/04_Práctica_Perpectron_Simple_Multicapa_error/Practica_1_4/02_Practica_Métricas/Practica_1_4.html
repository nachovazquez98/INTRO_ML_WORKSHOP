def MLP_binary_classification_2d(X,Y,net):
    plt.figure()
    for i in range(X.shape[1]):
        if Y[0,i]==0:
            plt.plot(X[0,i], X[1,i], '.r')
        else:
            plt.plot(X[0,i], X[1,i], '.b')
    xmin, ymin=np.min(X[0,:])-0.5, np.min(X[1,:])-0.5
    xmax, ymax=np.max(X[0,:])+0.5, np.max(X[1,:])+0.5
    xx, yy = np.meshgrid(np.linspace(xmin,xmax, 100), 
                         np.linspace(ymin,ymax, 100))
    data = [xx.ravel(), yy.ravel()]
    zz = net.predict(data)
    zz = zz.reshape(xx.shape)
    plt.contourf(xx,yy,zz, alpha=0.8, 
                 cmap=plt.cm.RdBu)
    plt.xlim([xmin,xmax])
    plt.ylim([ymin,ymax])
    plt.grid()
    plt.show()

Iris es el gnero de una planta herbcea con flores que se utilizan en decoracin. Dentro de este gnero existen muy diversas especies entre las que se han estudiado la Iris setosa, la Iris versicolor y la Iris virginica.

Las tres especies se pueden diferenciar en base a las dimensiones de sus ptalos y spalos. Se ha recopilado la informacin de 50 plantas de cada especie y se han almacenado en el archivo irisbin.csv. Dichas mediciones estn en centmetros junto con un cdigo binario que indica la especie a la que pertenece [-1, -1, 1] = setosa, [-1, 1, -1] = versicolor, [1, -1, -1] = virginica, se muestra la distribucin de los datos contenidos en el archivo. Se debe crear un programa capaz de clasificar automticamente los datos de 150 patrones usando un perceptrn multicapa. Es recomendable considerar 80% de los datos para entrenamiento y 20% para generalizacin.

Con la estructura optima de la red, se deben validar los resultados usando lo mtodos leave-k-out y leave-one-out con un perceptrn multicapa como clasificador. Se debe estimar el error esperado de clasificacin, el promedio y la desviacin estndar de ambos mtodos.

Conclusiones¶

En esta prctica pude profundizar sobre los diferentes mtodos de distribucin de los datos para poder evitar el sobre entrenamiento, en lugar de entrenar al modelo con el set de datos en su orden original. Ya que al dividir los datos aleatoriamente en subconjuntos para promediar sus diferentes resultados. En este experimento el LeaveKOut tuvo mayor precisin en las 3 clases del dataset. Cuando se realizan estas tcnicas el modelo adquiere unan mayor robustez, ya que se le al entrenar mayor variabilidad de los datos.

Un factor importante al elegir entre los mtodos de validacin cruzada k-fold y LOO es el tamao del conjunto de datos. Ya que si se llega a utilizar uno muy grande el algoritmo de repeated-k-fold puede ser mejor opcion, porque es menos costoso computacionalmente y las muestras de prueba son suficientes para tener un resultado significativo. Y Leave-P-Out con un tamao pequeo puede ser el ms apropiado, ya que se utilizarn ms muestras de entrenamiento en cada iteracin.

Adems se graficaron las curvas de prdida que nos brindan una mejor comprensin de cmo cambia el rendimiento del aprendizaje a lo largo de la cantidad de pocas y nos ayudan a diagnosticar cualquier problema con el aprendizaje que pueda conducir a un modelo inadecuado o excesivo.
